# 幹話產生器

可以選擇對哪一個職業說幹話 XD

## 如何開始

---

1. clone 專案
2. 開啟終端機(Terminal)到存放專案的資料夾
3. 輸入 npm install
4. 輸入 nodemon app.js
5. 開啟瀏覽器輸入http://localhost:3000

## 如何使用

---

1. 選擇要對工程師、設計師、創業家講幹話
2. 點選產生幹話
3. 幹話會出現在下方
